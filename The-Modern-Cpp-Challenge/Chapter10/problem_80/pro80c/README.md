pro80c
===============

Compressing and decompressing files to/from a ZIP archive  

Rewrite in C  with libzip


### Require
- libzip
https://libzip.org/  


### Buid and Run:    
% make all  

% cd bin    
% ./comp
> dir add: test1 
> file add: test1/sample1.txt 
> file add: test1/sample2.txt 
> file add: test1/sample3.txt 
> created: test1.zip
