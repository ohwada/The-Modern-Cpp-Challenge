problem_19
===============

Adding a range of values to a container 

Write a general-purpose function that can add any number of elements to the end of a container that has a method push_back(T&& value).  


### Buid and Run:  
% cd build  
% cmake ..  
% make  

% ./problem_19  
> 1 2 3 4 1 2 3 4
